Features

Built for AI. Secured by you.

Every card is sandboxed with strict limits. Your AI gets spending power — you keep full control.

🤖

AI-Native Payments

Let Claude or any AI assistant create and use virtual cards on your behalf via MCP.

🔥

Burner Cards

One-time use cards that auto-close after payment — zero lingering risk.

💰

Spending Limits

Cap every card so AI never overspends. Set monthly limits down to the dollar.

❄️

Instant Freeze

Pause any card in milliseconds. Resume when ready — you're always in control.

📊

Full Audit Trail

See every transaction your AI made, filterable by date, merchant, and status.

⌨️

CLI + MCP

Works from your terminal or directly inside AI tools via Model Context Protocol.

See it in action

Your terminal, your cards.

Create a card, make a payment, check the history — all without leaving the command line.

clawcard — zsh
# Create a burner card for a one-time purchase
$ clawcard burner 25.50 -m "domain.com"
✓ Burner card created — $26 limit, locked to domain.com
 
# Create a reusable card with a monthly cap
$ clawcard create -n "AI Tools" -l 50
✓ Card "AI Tools" created — $50/month limit
 
# Check recent transactions
$ clawcard transactions --since 7d
domain.com $25.50 settled 2 days ago
openai.com $20.00 settled 5 days ago
